# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## 🚀 Features

- Two-player gameplay (X vs O)
- Winner detection
- Draw match detection
- Restart game button
- Responsive design
- Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
